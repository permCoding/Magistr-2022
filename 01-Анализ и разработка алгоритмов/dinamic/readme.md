# Динамическое программирование  

---  

## Одномерная динамика  

сравнить рукурсию и динамику  

путь по лестнице  
путь к магазину
узнать кол-во путей  
узнать сам путь  
узнать минимальную цену пути  

---  

## Двумерная динамика  

минимальный путь где диагональная матрица  
путь в клеточном лабиринте  
о заполнении рюкзака
