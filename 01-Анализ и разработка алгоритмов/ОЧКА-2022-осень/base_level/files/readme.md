# Файлы  

неструктурированные файлы  
**txt** - список строк  

```txt
id,name,age
1,Света,27
2,Коля,22
```

структурированные файлы  
**csv** - плоские с примитивами  
**json** - иерархические данные  

Пример csv-файла:  

```csv
id,name,age
1,Света,27
2,Коля,22
```

Примеры json-файлов:  

```json
{
    "id": 1,
    "name": "Света",
    "age": 27
}
```

```json
[
    {
        "id": 1,
        "name": "Света",
        "age": 27
    },
    {
        "id": 2,
        "name": "Коля",
        "age": 22,
        "address": {
            "city": "Пермь",
            "street": "Ленина"
        }
    },
]
```
